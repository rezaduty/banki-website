راه اندازی پروژه
===================

راه اندازی سرور
------------


به محل فایل پروژه رفته و دستور زیر را اجرا میکنیم



.. Note:: . envsetup.sh && go run src/*.go

راه اندازی بانک ها
--------------

به محل فایل پروژه بر روی دستگاه رفته و دسوتر زیر را اجرا میکنیم


.. Note:: python bt.py -branch 2 -ip 192.168.1.12
